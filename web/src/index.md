# プライバシーポリシー

**最終更新日: 2026年7月6日**

本プライバシーポリシーは、Mezaru（以下「本アプリ」といいます）における利用者の情報の取り扱いについて定めるものです。本アプリをご利用いただく前に、本ポリシーをお読みください。

## 基本方針

本アプリは、指定した時間範囲の間、一定間隔でアラームを鳴らすことを目的としたアプリです。**本アプリは、利用者の個人情報を収集・送信しません。** アプリの設定内容はすべて利用者の端末内でのみ処理・保存されます。

## 収集する情報

本アプリは、氏名・メールアドレス・電話番号・位置情報・連絡先などの個人を特定できる情報を**一切収集しません**。

本アプリが端末内に保存する情報は、以下のアプリ設定に限られます。

- アラームの開始時刻・終了時刻・鳴動間隔などの設定
- アラーム音（通常音・最後の1回の音）の選択内容

これらの情報は利用者の端末内にのみ保存され、外部のサーバーや第三者に送信されることはありません。

## 情報の外部送信について

本アプリはインターネット通信を行う権限（`INTERNET` 権限）を保有しておらず、いかなる情報も外部へ送信しません。広告の配信、アクセス解析ツールの利用、第三者へのデータ提供のいずれも行いません。

## 端末の権限について

本アプリは、アラーム機能を提供するために以下の権限を使用します。これらは端末上での動作のために利用されるものであり、個人情報の収集を目的とするものではありません。

- **正確なアラーム（`SCHEDULE_EXACT_ALARM` / `USE_EXACT_ALARM`）** — 指定した時刻に正確にアラームを鳴らすために使用します。
- **通知の表示（`POST_NOTIFICATIONS`）** — アラームの稼働状態や発火を通知するために使用します。
- **全画面通知（`USE_FULL_SCREEN_INTENT`）** — ロック画面でもアラーム画面を全画面表示するために使用します。
- **画面の点灯・ロック解除（`WAKE_LOCK` / `TURN_SCREEN_ON` / `DISABLE_KEYGUARD`）** — アラーム発火時に画面を点灯させるために使用します。

## 情報の保存期間と削除

アプリ設定は利用者が変更・削除するまで端末内に保存されます。本アプリをアンインストールすると、保存されたすべての設定情報は端末から削除されます。

## 児童のプライバシー

本アプリは特定の年齢層を対象としたものではなく、個人情報を収集しないため、児童から個人情報を取得することはありません。

## プライバシーポリシーの変更

本ポリシーの内容は、法令の変更やアプリの機能追加に応じて予告なく変更されることがあります。変更後のポリシーは本ページに掲載した時点で効力を生じるものとします。
